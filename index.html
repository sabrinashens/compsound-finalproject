Beats@Root+Chord+Scale'Inversion([Loop])

E.g., 1@C9'1, 2@Adim9[2]


❒ Beats: ℤ@ (ℤ as a positive integer)

Recommended: 4@, 2@, 1@ (whole, half, quarter)


❒ Root: C, C#, D, D#, E, F, F#, G, G#, A, A#, B


❒ Chord: \ , m, dim, aug

major, minor, diminished, augmented


❒ Scale: 1, 3, 5, 7, 9

(root),

(root, 3rd),

(root, 3rd, 5th),

(root, 3rd, 5th, 7th),

(root, 3rd, 5th, 7th, 9th)


❒ Inversion: \ or 'ℤ

'1: (root, 3rd, ...) => (3rd, ..., root + an octave)

'2: (root, 3rd, ...) => (..., root + an octave, 3rd + an octave)

...


❒ Loop: \ or [ℤ]
