Music Theory Utilized

1. Beats

4 as wholenote, 2 as halfnote, 1 as default quarter note.

2. Root Key

pick one from an octave (C4-B4)

3. Chord choice

Major, minor, diminished, augmented. And how they differ

4. Scale

Root, ... , nth factor

5. Inversion

The inversion is made by moving the first or the first few elements up by an octave

For example, if inversion = 1, then just move the root key up by an octave

If 2, then move the root and the 3rd note up by an octave

6. Loop

simply how many repetitions


How Command is Processed

If beats inputed

Beats: split by @, the first element of the array

If loops inputed

Loops: split by [, the second element of this array, with "]"removed

If inversion inputed

Inversion: split by ', the second element of this array

Optional features: Beats, Loops, and Inversion are either default or obtained

What left is required: Root, Chord, Scale

The last character of the remaining is the scale number

using a math formula to convert the scale into a number of notes

slice the scale number -> The root key + chord choice

Take the root key and map it onto the pitch

With the root pitch, extend it to the corresponding chord-scale according to its chord

As we get the basic chord-scale, we start doing inversion

Take the number of inversion as x, we take the values of first x element(s), then pitch+12 add to the end

Then slice out the first x elements

What is known now: the inverted or basic chord-scale with loops and beats if given

Find each note the start time and endtime. The first note starts by the start time 0

and the end time is determined by beats/tempo.

diff between beats and tempo: tempo is the same for all notes, beats differ each note's length

The greater beats is, the note plays slower

The greater tempo is, the entire melody plays faster

Then the notelist with note pitch, start time, and end time, which would be ready to play with osc
